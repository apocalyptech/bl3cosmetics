Borderlands 3 Cosmetics Galleries

This is a collection of galleries which show off all available cosmetic items for Borderlands 3, including ones which can't officially be acquired in-game. These should be complete as of the August 5, 2021 patch (which introduced Vault Card 2).

This site doesn't really intend to give much more information than just the galleries themselves. This Google Sheet contains more information such as drop sources for many of the cosmetics, which ones are locked to particular DLCs, and "Balance" paths which can be used in conjunction with writing hotfix mods.

If you want to just unlock all cosmetics with an external utility, there's a few utilities which will do that for you. Note that vehicle skins are unlocked on a per-save basis, whereas all other cosemtics are stored in the global profile.sav file.

  • Raptor's Web-Based Editor - Upload directly there. Sourcecode is also available.
  • FromDarkHell's Profile Editor - Download and run locally (does not unlock vehicle customizations)
  • apocalyptech's CLI editor - Python-based and commandline, so probably not for many folks.

If you have any comments/questions about any of this, feel free to post in this thread at the Gearbox forums, or submit an Issue/PR on the github repo. Sourcecode for the site generation, and all the images themselves, are also available in the github repo, so feel free to clone that and host it anywhere else, if you like.
