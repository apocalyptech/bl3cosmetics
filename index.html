Borderlands 3 Cosmetics Galleries

This is a collection of galleries which show off all available cosmetic items for Borderlands 3, including ones which can't officially be acquired in-game. These should be complete as of the August 5, 2021 patch (which introduced Vault Card 2).

This site doesn't really intend to give much more information than just the galleries themselves. This Google Sheet contains more information such as drop sources for many of the cosmetics, which ones are locked to particular DLCs, and "Balance" paths which can be used in conjunction with writing hotfix mods.

If you want to just unlock all cosmetics with an external utility, there's a few utilities which will do it for you:

  • Raptor's Web-Based Editor - Upload `profile.sav` directly there. Sourcecode is also available.
  • FromDarkHell's Profile Editor - Download and run locally
  • apocalyptech's CLI editor - Python-based and commandline, so probably not for many folks
